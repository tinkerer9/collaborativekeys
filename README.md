# CollaboKeys
a collaborative keyboard game

Made by @tinkerer9, @LethalShadowFlame, and @chickenlloyd for a hackathon with theme "Connection".

Players collaborate to play any keyboard-based game with their assigned key. Think quick!

## Dependencies
- `nodejs`
- `http`
- `fs`
- `path`
- `socket.io`
Made for MacOS.
